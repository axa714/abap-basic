CLASS LCL_EVENT_HANDLER IMPLEMENTATION.
METHOD ON_HOTSPOT_CLICK.

READ TABLE GT_DATA INTO GS_DATA INDEX ES_ROW_NO-ROW_ID.
SELECT SINGLE * FROM SCUSTOM INTO SCUSTOM WHERE ID = GS_DATA-CUSTOMID.

IF E_COLUMN_ID-FIELDNAME = 'CUSTOMID'.
CALL SCREEN 110 STARTING AT 10 10.
ENDIF.
ENDMETHOD.

METHOD ON_TOOLBAR.
DATA: LS_BUTTON TYPE STB_BUTTON.
